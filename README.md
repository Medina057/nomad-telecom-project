# Nomad Telecom project
### Self-Service подключение тарифов и услуг


## О проекте

Этот проект представляет собой кейс для портфолио, демонстрирующий навыки бизнес-анализа и системного анализа.

В проекте описывается процесс подключения мобильных тарифов через приложение самообслуживания, интегрированное с CRM, биллинговой системой и SMS-шлюзом.

---

## Цель проекта

Разработать бизнес и системную документацию для процесса подключения тарифа, включая функциональные требования, бизнес-процессы, UML-модели и спецификацию REST API.
