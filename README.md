# Self-Service подключение тарифов и услуг pet-project
### Nomad Telecom (вымышленная компания)


## О проекте

Полный аналитический пакет для перевода процесса подключения тарифов у телеком-оператора из контакт-центра/салона связи в самообслуживание через мобильное приложение — от бизнес-цели до готового к разработке API-контракта.

Nomad Telecom — вымышленная компания. Процесс смоделирован на основе общедоступной практики рынка телеком-операторов Казахстана.

---

## Цель проекта

Разработать бизнес и системную документацию для процесса подключения тарифа, включая функциональные требования, нефункциональные требования бизнес-процессы, UML-модели, Модель данных и спецификацию REST API.

## Ключевое архитектурное решение, отражённое в проекте

Активация услуги в биллинге спроектирована как асинхронная операция (202 Accepted + requestId, затем отдельный webhook с финальным результатом), а не синхронный запрос-ответ — это осознанное решение зафиксировано в SRS, отражено в Sequence-диаграмме и в OpenAPI-спецификации через отдельный блок webhooks.
